# Portfolio - Danie Chey

Portfolio personnel de développeuse web et designer.

## Technologies

- React 18
- Vite 5
- React Router DOM 6
- CSS3

## Installation

```bash
npm install
npm run dev
```

## Projets présentés

### Développement Web

- **Ohmyfood** - Intégration mobile-first avec animations CSS
- **Kasa** - Application React de location immobilière
- **Sophie Bluel** - Interface dynamique connectée à une API REST
- **Kampoy** - Application e-commerce

### Design & Formation

- **Alimentation** - Module de formation interactif
- **Coach Linguistique** - Plateforme de coaching en ligne
- **Énergies Renouvelables** - Module de formation interactif
- **Recherche Web** - Simulation de recherche web
- **Cubyn** - Supports de communication et recrutement
